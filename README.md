# friend-of-friend-network
# 🌿 Graph Growth Models: Preferential Attachment & Copying

This repository contains Python implementations of generative network models used to simulate and analyze the evolution of complex graph topologies, specifically focusing on tree structures, preferential attachment, and uniform copying mechanisms.

## 🚀 Key Features

* **Preferential Attachment Models (PAM):** Simulates scale-free network growth where new nodes prefer linking to highly connected nodes.
* **Uniform Copying Models (UAM):** Replicates network duplication dynamics where new nodes copy links from existing nodes uniformly at random.
* **Optimization & Building:** Code architectures dedicated to accelerating graph building and optimizing structural constraints.
* **Graph Exporting:** Built-in support to save generated network structures to the standardized `.graphml` format for external visualization (e.g., in Gephi).

## 🛠️ Tech Stack

* **Language:** Python 3
* **Graph Processing:** NetworkX (or graph-tool depending on implementation)
* **Mathematical Operations:** NumPy
